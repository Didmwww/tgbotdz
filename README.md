
#  Student Services Telegram Bot

Цей проект — це Telegram-бот, розроблений для зручного прийому та обробки замовлень на виконання студентських робіт (реферати, презентації, лабораторні та курсові). Бот автоматизує процес спілкування з клієнтом: від збору вимог до завдання до підтвердження оплати та видачі готової роботи.

##  Основні можливості

###  Для студентів (Користувачів)
* **Зручне меню замовлень:** Вибір типу роботи із заздалегідь визначеним прайсом (Реферати, Лаби, Презентації тощо).
* **Покрокова подача заявки:** Використання машини станів (FSM) для збору даних про курс, предмет та саму умову завдання (з підтримкою тексту, фото та файлів).
* **Система черги:** Клієнт бачить свою позицію в загальній черзі на виконання.
* **Інтеграція оплати:** Після підтвердження завдання адміном, користувач отримує посилання на банку Monobank для оплати та форму для підтвердження переказу.

###  Для Адміністратора
* **Модерація замовлень:** Адмін отримує сповіщення про нові заявки (з деталями та файлами) і може прийняти їх або відхилити за допомогою Inline-кнопки.
* **Контроль оплат:** Після підтвердження оплати клієнтом, адмін отримує ПІБ відправника для перевірки виписки в Monobank і фінально підтверджує взяття в роботу.
* **Пряма відправка результатів:** Вбудована команда `/send` дозволяє адміну відправити готовий файл або текст безпосередньо в чат конкретному студенту за його Telegram ID.

##  Технологічний стек
* **Мова програмування:** Python
* **Фреймворк:** [aiogram 3.x](https://docs.aiogram.dev/)
* **Архітектура:** FSM (Finite State Machine) для керування діалогами

##  Встановлення та локальний запуск

1. **Клонуйте репозиторій:**
   ```bash
   git clone [https://github.com/ВашЛогін/НазваРепозиторію.git](https://github.com/ВашЛогін/НазваРепозиторію.git)
   cd НазваРепозиторію



2. **Створіть віртуальне середовище (рекомендовано) та встановіть залежності:**
```bash
python -m venv venv
source venv/bin/activate  # Для Windows: venv\Scripts\activate
pip install aiogram

```


3. **Налаштуйте конфігурацію:**
Відкрийте файл `botdz.py` та заповніть наступні змінні (рекомендується винести їх у змінні середовища):
* `TOKEN` — токен вашого бота від @BotFather.
* `ADMIN_ID` — ваш особистий Telegram ID для отримання замовлень.
* `JAR_URL` — посилання на вашу банку Monobank для оплат.
* `ADMIN_USERNAME` — ваш @username для зв'язку.


4. **Запустіть бота:**
```bash
python botdz.py

```



##  Команди бота

* `/start` — Запуск бота та виклик головного меню.
* `/send` — (Тільки для адміна) Команда для надсилання готової роботи студенту. Бот попросить ввести ID студента, а потім прикріпити файл.

##  Логіка системи черги

Бот веде тимчасову (оперативну) чергу замовлень `QUEUE` у пам'яті.

* Студент додається у чергу при відправці завдання.
* Студент видаляється з черги: при скасуванні адміном, відсутності оплати, або після успішної видачі готової роботи через команду `/send`.
*(Увага: при перезапуску скрипта `botdz.py` поточна черга обнуляється, оскільки дані зберігаються у списку, а не в базі даних).*
